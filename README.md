# About the Project

## Career Prediction System Using Machine Learning

The Career Prediction System is a Machine Learning project developed to predict suitable career paths for students based on their academic performance, subject-wise marks, stream, and overall percentage. The project aims to demonstrate how machine learning techniques can be applied to educational data to provide career recommendations.

The system uses students' marks in subjects such as Physics, Chemistry, Mathematics, Biology, English, Computer Science, Economics, Accountancy, Business Studies, History, Geography, and Political Science, along with stream information and aggregate percentage, to predict a potential career category.

### Key Features

* Data preprocessing and cleaning
* Missing value analysis
* Label encoding for categorical variables
* Feature scaling using StandardScaler
* Train-test data splitting
* Career prediction using Logistic Regression
* Model evaluation using:

  * Accuracy Score
  * Classification Report
  * Confusion Matrix
* Confusion Matrix Heatmap visualization
* Interactive user-input based prediction system
* Decoding predicted career labels into human-readable career names

### Technologies Used

* Matplotlib
* Seaborn

### Machine Learning Workflow

1. Data Collection

### Project Objective

The primary objective of this project is to explore how machine learning can assist in career guidance by analyzing students' academic performance and identifying patterns that may indicate suitable career paths.

### Learning Outcomes

Through this project, the following machine learning concepts were implemented and understood:

* Classification Problems
* Logistic Regression
* Data Preprocessing
* Feature Scaling
* Model Evaluation Metrics
* Confusion Matrix Analysis
* Heatmap Visualization
* Real-world Prediction Systems


This project serves as an end-to-end Machine Learning implementation that demonstrates the complete workflow from data preprocessing to prediction and model evaluation.
